# 背景表

背景表决定了你的角色从哪里来，以及在游戏开始前你的角色做了什么。你根据你的年龄表的投掷结果，来投掷童年表和一定数量的人生经历表。人生经历表可能会让你投掷其他表来确定你经历了什么。
