# 起始装备

游戏开始时，你将使用基于你起始生活水平和职业的装备。为了确定你的起始生活水平，投3d6，并添加你的经验表获得的所有改动。然后在起始生活水平表中找到总数(核心书中描述，第100页)。除了装备，你的角色开始有一个有趣的东西(影子，第26页)。
