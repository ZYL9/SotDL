# 忠诚士卒Loyal Soldiers

帝国境内的紧张局势正不断加剧，各领、各类组织和有钱人都投入了大量资金来建设军队。如今，人们可以在洲内的几乎每一处地方发现军队的存在。神圣王国Holy
Kingdom在凯拉克斯Caecras边境集结了大军，而德鲁奇Drudge（按：即兽人之王）也照此行动以准备入侵这个受憎的王国。在提尔Tear境内肆虐的野兽人受到了不计其数的民兵的抵抗，他们为了自己的社区而战；而位于北境Northern
Reach的圣战邦Crusader
States则总是寻求新成员。作为军事组织，忠诚士卒们会在冒险间隔前往他们被调往的任何地方，他们可能会发现自己正在进行与更大的战争力量有关的冒险。

## 小队战术Squad Tactics

在每次冒险开始时，你们的团队将才能池充满，才能池以指示物表示。你们的团队拥有3+团队等级一半的指示物。每当一个小组成员将对另一个小组成员触及范围内的目标进行攻击掷骰时，第二者可以消耗任意数量的指示物，以使此次掷骰获得等量的优势。此外，小组成员可以消耗一个指示物来协助help短程范围内的另一个小组成员而不需要使用动作。
