# **魔王千面ASPECTS OF THE DEMON LORD**

面貌条目描述了最盛行的信仰和邪教信徒的常见行为。也包含与面貌相关的传承。当你从你的道途中获得腐化时，掷d6，并查看你所信奉的面貌下的表格，看看它是如何显现的。如果你获得了一个你已经拥有的结果，重新掷骰。

最终，当你使用从你的道途获得的恶魔召唤法术时，所出现的恶魔将获得所描述的天赋以代替恶魔天赋选项。
