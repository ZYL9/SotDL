# 伤害和治愈

### 伤害和治愈

有时，一次尝试会导致玩家受到伤害或者可能会治愈伤害。大多数这样的游戏效果已经定义好了:法术描述、武器词条、天赋描述和生物数据都会告诉你在什么情况下掷什么骰子来造成伤害。然而，有时候，你也会不知道该造成多少伤害。

查看**伤害基线表**来决定伤害，不仅是造成的伤害，还有治愈的伤害。"未受限"列指的是持续性效应，例如：着火，浸入酸液池。使用"受限"列来作为单次效应的伤害，例如：塌方，爆炸，治疗药水的治疗。

### 伤害基线表

  团队等级   未受限   受限
  ---------- -------- -------
  新手       1d6      1d6+2
  入门       1d6+1    2d6+1
  专家       2d6      4d6
  大师       4d6      8d6
