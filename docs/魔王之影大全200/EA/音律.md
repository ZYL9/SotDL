# 音律

#### 漆黑圣歌之音符NOTES OF THE BLACK HYMN

音律传承 攻击 5环

------------------------------------------------------------------------

**施法需求**你必须至少有1点腐化

**效应范围** 短程范围内所有能听见你的生物

------------------------------------------------------------------------

空气中充斥着一阵阵噪音，迫使每个生物进行意志掷骰。生命值为10或更低的目标自动失败。若目标的生命值超过20点，每10点都会提供1点优势。若成功，目标会变成一个与其体型相当的怪物，只要你保持专注就能驱使它们去杀人，至多1分钟
