# 幻术

#### 心之所愿HEART'S DESIRE

幻术传承 实用 2环

------------------------------------------------------------------------

效应范围 短程范围内一个可见生物

持续时间 1小时

------------------------------------------------------------------------

你幻化成目标最为渴望的人。但幻像不足以掩盖体型和运动方式上的差异，这种不协调感可能会促使目标试图辨别幻像（见核心书）。直到效果结束前，你对目标的社交场合掷骰有2点优势

#### 感官密牢OUBLIETTE

幻术传承 攻击 4环

------------------------------------------------------------------------

效应范围 中程范围内的一个生物

------------------------------------------------------------------------

目标的视觉和听觉得到减弱。对抗目标的感知进行智力掷骰。若目标生命值大于等于50则你有1点劣势。成功时目标陷入持续1分钟的目盲、耳聋和束缚状态

**攻击掷骰20+** 状态会持续1个小时
