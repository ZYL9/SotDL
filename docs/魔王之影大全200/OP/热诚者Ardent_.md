# 热诚者Ardent

念动魔法授予那些发现其威能的人仅凭意念影响周遭世界的手段。他们可以移动物件，向敌人投掷念动力并凭空而行。热诚者掌握念动魔法以补充自己的搏斗能力。通过集中意念，他们可以击穿可能面临的任何抵抗，并偏转来袭的攻击。

### 7级热诚者

属性：任选三者提升1

特征：生命值+3，威能+1

语言和专业：你可以听说一门新语言或获得一门新专业

施法：你探索念动传承或学习一个新念动法术

热诚打击Ardent Strike：在你通过使用集中精神Focus
Mind天赋而使精神集中时，你使用近战武器或投掷武器进行的攻击造成1d6点额外伤害。此外，如果你的攻击掷骰的总数为20或更高并超过了目标数字至少5点，目标还会受到1d6点额外伤害。

### 10级热诚者

特征：生命值+2

施法：你学习一个新法术。

反应护盾Reactive Shield：在你通过使用集中精神Focus
Mind天赋而使精神集中时，你可以使用一个反应提前结束你的集中状态并降低你将要受到的伤害。你的每点威能值都能降低1d6点伤害。

预备集中Ready Focus：在你的回合内，你可以用一个反应使用集中精神天赋。
