# 牧树人Tree Shepherd

牧树人们保护着原始森林与林地，使其免受非法砍伐者与纵火者攻击。他们是任何破坏自然的人坚定的敌人，能够唤醒他们保护的森林，使树木响应号召并肩作战。然而，牧树人所使用的的魔法也改变了他们自身，促使他们舍弃了血肉之躯，变得更像他们守护的森林。

### 7级牧树人

属性：任选三者提升1

特征：生命值+5，威能+1

语言和专业：你可以听说一门新语言或获得一门新专业

施法：你探索自然传承或学习一个新自然法术
