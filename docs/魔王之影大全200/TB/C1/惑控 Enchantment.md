# 惑控 Enchantment

## 极乐术 Bliss

惑控 攻击 3环

------------------------------------------------------------------------

效应目标：短程范围内任意数量的生物

------------------------------------------------------------------------

每个目标必须进行一次意志检定。如果目标被魅惑，进行此检定时获得1点劣势。如果你在上次休息之后的任一时刻攻击了目标，那么目标对此法术免疫。如检定失败，目标陷入眩晕dazed状态1分钟或直到其受到伤害。
