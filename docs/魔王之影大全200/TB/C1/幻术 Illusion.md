# 幻术 Illusion

## 志趣相投 Kindred Soul

幻术 实用 1环

------------------------------------------------------------------------

持续时间：1小时

------------------------------------------------------------------------

持续时间内，你可见的生物中所有体型与你种族通常体型一样、或是在你种族通常体型区间内的生物，将把你认作同族。此外，每当你说话时，至少懂得一种语言的生物都可以理解你说的话，就如同你在用一种他们所知的语言一样。
