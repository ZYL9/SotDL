# 起始装备 Starting Equipment

你的角色开始游戏时拥有基于其起始生活水平和他们专业的装备。投掷3d6并基于人生经历表获得加值或减值，以此来决定你的起始生活水平。参照起始生活水平表来查看结果（核心书100页）。此外，你的角色起始额外拥有一个有趣的小玩意儿（核心书26页）
