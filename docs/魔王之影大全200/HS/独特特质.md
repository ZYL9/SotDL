# 独特特征

许多气灵的皮肤上有着有趣的图案和特征，这些特点有助于区分辨认他们。某些气灵的皮肤上似乎有着银色的花纹或闪闪发光的小圆点，或者他们的肤色可能会随着环境变化而变化。为你的角色外观增加更多的细节，例如加上一些小疤痕和破损。下表会让你的橘色获得一个有趣的身体特点，或者给予你灵感让你想出一些不一样的东西。
