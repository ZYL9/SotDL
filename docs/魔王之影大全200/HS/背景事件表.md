# 背景事件

在你的角色开始第一次冒险之前，角色之前的人生经历可以告诉你，你角色的世界观是怎么样的，以及他或她是如何应对挑战的。角色可能曾建立过许多段关系后又失去了这些关系，捡到了意外之财后遭受挫折，甚至可能还有过诡异的经历。下表提供了一种生成人生重要事件的方法。你可以掷骰子随机得到表中的一个结果或者可以写一个属于你自己的背景事件，用该表激发你的灵感。
