# **起始装备Starting Equipment**

你的角色得到与你的起始生活水平和专业相关的装备开始游戏。为了确定你的其实生活水平，掷3d6，根据你的人生经历表掷骰的指示获得加值或减值。依照生活水平表参考结果（生活水平在规则书100页中描述），然后获得如下表所示的起始装备。除了你的生活水平所赋予的装备之外，你的角色起始额外拥有一个有趣的小玩意儿（核心书26页）。
