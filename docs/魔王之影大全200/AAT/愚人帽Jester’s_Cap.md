# 愚人帽Jester’s Cap

愚人帽在利杰出现的三天内，整座城市都被混乱笼罩。这件遗物古怪魔法造成的影响直到现在仍可在它被使用的地方附近扭曲的建筑中看到，而捕鼠人们至今仍在与下水道中繁殖壮大的蓝色老鼠战斗。此外，依然没人能拿到找回阿斯特丽德雕像的赏金，这尊雕像在迎来一位流浪小丑的参观后，主动从阿斯特利德女士大教堂Lady
Astrid’s Cathedral的基座上离开了。

这顶魔法帽子来自何处?位于凯克拉斯奥能塔的大人物声称这件遗物是一位疯法师的作品，那个疯子认为可以将混乱魔法束缚在帽子中以操控这股力量。奥能塔的法师认为，如果所谓的“奇迹之城”可以管控城内魔法的滥用，而不是一直让它像建城以来那样处于自由状态，整件事本不会发生。尽管奥能塔发出了这样的指控，但人们普遍认为这顶帽子是被带到利杰的，因为其他地方后来也不幸遇到了疯狂的小丑和他的魔法帽子，例如一个村庄因为给小丑提供住宿而得到了一条时髦的皮短裤，或是农夫汉姆的谷仓里突然多出来的两百只鸽子作为回报。 

事实上，“愚人帽”的存在要归功于双面的狂欢之神Revel，在一个凡人没能好好地欣赏他的一个笑话后，他气冲冲地创造了这顶帽子。狂欢之神将那个年轻女人的情人变成了一头“十分有劲”的公牛。他觉得这件事很幽默，并乐在其中，但女人对此不感兴趣，要求神赶紧将她的情人恢复原状。狂欢之神此时罕见的表现出了懊悔，并把男人恢复了原形，随后就沮丧的离开了。路上，他一边大口的喝酒，一边抱怨着那个女人的指责，他越喝越感到烦躁。突然，他有了一个主意，想出了一堆恶作剧玩具和荒诞闹剧的点子，并将每一个点子的精髓注入了一顶缀有一串铃铛的鲜艳帽子中。他将这顶帽子丢进了凡世，为每一个戴上它的人带来欢笑。如果他们觉得从帽子里抽出的第一件东西不太好玩，那么下一份礼物可能会更加有趣。

自从“愚人帽”进入凡世以来，它一直在制造着混乱。这顶帽子迫使其佩戴者从帽子里抽出“滑稽”的物品并释放它们。从帽子里出来的大多数东西只是些小麻烦，但偶尔也会出现一些可怕的东西，给附近的每个人都带来大麻烦。

 

**愚人诅咒Jester****’****s Curse**
：如果你戴上“愚人帽”，你会立即受到相当于8环法术的诅咒。在被诅咒时，你无法摆脱这顶帽子。如果你丢弃它，它会在1d6轮后回到你的头上。此外，除非开始发疯，否则你无法以任何方式减少疯狂。此外，在每次休息之间，你必须至少使用一次该物品。如果你没使用，你会受到1d6点的生命减值，直到你再次使用它。如果你在被诅咒时发疯，诅咒会被解除，遗物会立刻消失，然后出现在GM选择的某个地方。

**来点乐子Something Funny?：**
如果你受到“愚人诅咒”，你可以使用一个次要动作，摘下帽子，伸手进去，从里面拿出一件事物。你获得1点疯狂，但你不会因此而陷入恐慌。投2d20并参照下表来确定从帽子里出现的东西。一旦使用了此天赋，你必须等待至少1轮才可以再次使用。 

**厄运数字Unlucky Number**
：如果你受到“愚人诅咒”，则你每次完成休息时都要掷一个d20，并记录结果。在你完成下一次休息前，这个数字将作为你的厄运数字。每当你掷d20时，如果出现了你的厄运数字，你获得2d6点疯狂。
