# 炼金 Alchemy\*

## 毒素飞弹 Venomous Dart

炼金 攻击 0环

------------------------------------------------------------------------

效应目标 短程范围内一个生物或物件

------------------------------------------------------------------------

你向目标发射一枚灌注魔法毒素的飞弹。进行一次有1点优势，对抗目标防御的智力攻击掷骰。若成功，飞弹命中，对目标造成1d3点伤害。目标必须进行一次成功的力量检定掷骰，若失败则陷入1分钟中毒状态。

攻击掷骰20+ 目标进行的力量检定掷骰获得3点劣势。
