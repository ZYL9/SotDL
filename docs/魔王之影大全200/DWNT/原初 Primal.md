# 原初 Primal

## 泌毒尖牙 Venomous Fangs

原初 实用 1环

------------------------------------------------------------------------

持续时间 1分钟

------------------------------------------------------------------------

你的牙齿伸长变为毒牙，视为拥有灵巧属性的天生武器，造成1d6+1点伤害。当你使用毒牙进行的攻击掷骰成功时，目标必须进行一次有1点劣势的力量检定。若失败，将受到1d6点伤害，并陷入1分钟中毒状态。如果目标已经中毒了，则改为受到1d6点额外伤害。
