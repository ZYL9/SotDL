# 真名伟力Power in a Name

狂龙Kulo翻译

魔王之影的染毒之页

真名是一个隐秘之词，既赋予、亦表达了它所属于的事物或存在的本质。真名赋予其形并反映着本质。没有真名，生物或物件就会解体，回归到巨灵造物的原始混沌之中。由于真名对存在是如此不可或缺，通晓真名会获得对真名指向的目标的相当大的控制权。仅仅说出真名并不会产生任何作用，但通过魔能的注入，说出真名者可以召唤出真名指向的生物或物件，迫使它服从自己的命令，或者通过部分或完全解放真名来毁灭真名指向的事物。

巨灵们说出了创造宇宙的权能之语word of
power，但也认知到这种创造神言会对他们的造物带来可怕的混乱。因此，在创造了世界之后，他们把这种语言的知识深藏起来，从现实中抹去了几乎所有关于它存在过的迹象。因此，大多数人并不知道他们的真名是什么，更不用说他们根本就不知道自己拥有真名。然而，尽管巨灵采取了种种手段，但几个世纪之后，这些语言的痕迹还是被发现了；当魔法使用者们施法的时候，他们会高声说出一些音节、单词、甚至短语。
