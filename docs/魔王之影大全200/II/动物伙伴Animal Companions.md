# 动物伙伴Animal Companions

世界各地的各种动物都可以被捕捉并训练，用来执行各种任务。这些小伙伴们被训练成同伴，可以用来代步、被教导与你一同作战、甚至可以用来负载装备。每种动物的属性框都可以在主规则书中找到。稀有程度根据地区，由GM决定。
