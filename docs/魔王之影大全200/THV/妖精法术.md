# 妖精法术

## 脱皮术SHED THE SKIN

妖精传承  攻击法术  5环

------------------------------------------------------------------------

施法需求 你必须有1点或更多的腐化

效应范围 全身覆盖有皮肤的活物

------------------------------------------------------------------------

对抗目标力量进行智力攻击掷骰。如果目标的生命值为20或更低，你将自动成功。如果目标生命值为40或更高，你的攻击掷骰有1点劣势。若成功，目标的皮肤从体表脱落，使其变成一个血骸，失去所有原本的数据。

献祭 你可以消耗一次该法术的施法次数来施放妖精传承中的幻象术。
