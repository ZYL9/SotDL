# 毁灭法术

## 祈求无名者INVOKE THE NAMELESS ONE

毁灭传承  攻击  1环

------------------------------------------------------------------------

施法需求 你必须拥有疯狂奉献天赋，且属于无名众教派

持续时间 1分钟

------------------------------------------------------------------------

你受到2点伤害。当你施法时和每轮结束时，一股毁灭性的力量脉冲从你体内涌出，直到法术结束。从你的空间中的一个点向外扩散至1d6码半径，对你造成1点伤害，并对区域内的所有事物造成1d6点伤害，生物可以进行敏捷掷骰，成功则不受伤害。
