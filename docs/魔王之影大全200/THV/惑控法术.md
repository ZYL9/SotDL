# 惑控法术

## 苦痛欢愉PLEASURE FROM PAIN

惑控传承  攻击法术  3环

------------------------------------------------------------------------

施法需求 你必须有1点或更多的腐化

效应范围 中程范围内一个你可视的活物

------------------------------------------------------------------------

你扭曲目标的感觉，使其在感受到痛苦时反而感到强烈的快感。对抗目标意志进行智力攻击掷骰。如果目标有6或更多的疯狂，你的攻击掷骰有1点优势。若成功，在1分钟内，每当目标受到伤害时会受到1d6点额外伤害，并且在1轮内攻击和检定掷骰带有1点优势，以此种方式获得的优势可以叠加。

攻击掷骰20+ 目标同时陷入魅惑，直到效应结束。
