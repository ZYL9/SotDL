# 诅咒法术 Curse Spells

### 墓王诅咒 Curse of the Tomb Kings

诅咒 攻击 3环

------------------------------------------------------------------------

效应目标 中程范围你一可见活物

------------------------------------------------------------------------

进行一次对抗目标力量的智力攻击掷骰和一次对抗目标意志的智力攻击掷骰。若对力量攻击成功，则目标被剧烈的疼痛折磨，受到2d6点伤害。若对意志攻击成功，则目标陷入1分钟恐慌状态。若两者均成功，则目标被诅咒。因此被诅咒的目标处于力竭状态，且无法治愈伤害。
