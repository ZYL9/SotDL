# 星光法术 Celestial Spells

### 艳阳锤 Sun Hammer

星光 攻击 4环

------------------------------------------------------------------------

效应范围
极远范围内一处被阳光照射的地面一点为圆心，2码半径，25码高的垂直柱状区域。

------------------------------------------------------------------------

一道炽烈的阳光射线对该范围内所有事物造成2d6+3点伤害。范围内每个生物必须进行一次力量检定。若失败则倒地并陷入1分钟力竭状态。若成功则受到的伤害减半。
