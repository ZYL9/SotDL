# W

## 低语仙人掌 Whisper Cactus

低语仙人掌成堆地生长在荒芜之地边缘，深红色的花在手指般长的刺之间，这说明了它们的危险。当任何事物移动到它们的感知范围内，它们会用巨力将足以贯穿钢铁的针刺喷射而出。随着喷射，喷出的针刺会产生模糊的低语声，这些植物也因此而得名。
