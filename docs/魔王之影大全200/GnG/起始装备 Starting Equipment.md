# 起始装备 Starting Equipment

所有人类角色开始游戏时拥有基于其起始生活水平和他们专业的装备。投掷3d6并基于人生经历表获得加值或减值，以此来决定你的起始生活水平。参照起始生活水平表来查看结果（核心书100页）。此外，你的角色起始额外拥有一个有趣的小玩意儿（核心书26页）

## 起始生活水平 Starting Lifestyle

  3d6     生活水平
  ------- --------------------
  3-      一贫如洗 Destitute
  4-5     贫困 Poor
  6-12    糊口 Getting By
  13-15   舒适 Comfortable
  16-17   富裕 Wealthy
  18+     阔绰 Rich
