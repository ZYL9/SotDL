# 外貌表

## 年龄

  3d6     年龄
  ------- ------------------------------------------------------------------------
  3       你是一个6岁或更小的孩子。将你的体型、身高和体重减半。
  4-5     你是一个青少年，7到10岁。将你的身高和体重降低25%。投掷一次人生经历表。
  6-8     你是一个年轻人，11到25岁。投掷两次人生经历表。
  9-12    你是一个成年人，26到40岁。投掷四次人生经历表
  13-15   你是一个中年人，41到50岁。投掷五次人生经历表。
  16-17   你是一个老年人，51到75岁。投掷六次人生经历表。
  18      你是一位长者的成年人，76岁或以上。投掷七次人生经历表。

 

## 体格

此表可修改您的起始身高和体重。除了霍布和螨虫之外，所有地精的起始高度都是36英寸，体重是60磅。螨虫的起始高度为18英寸，重量为15磅，而霍布的起始高度为48英寸，重量为100磅。

  3d6     体格
  ------- -----------------------------------------------------------------------
  3       你很小。将你的身高降低3.46英寸，体重每英寸降低2磅
  4-5     你太矮了。把你的身高减少2.6英寸，体重每英寸减少2磅。
  6-8     你很瘦。减重36磅。
  9-12    你的身高和体重一般。
  13-15   你很重。你的体重增加5d6磅。
  16-17   你很高。身高增加2.56英寸，体重每英寸增加2磅。
  18      对于一个地精来说，你是相当大的。身高增加3.46英寸，体重每英寸增加3磅。
